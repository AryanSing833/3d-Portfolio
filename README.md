## 🌐 <b><samp>About This Project</samp></b>

A **3D interactive portfolio** built with **React 19, Three.js, GSAP, and Tailwind CSS v4**. It showcases my projects, skills, achievements, and contact information — all in an animated, scroll-driven experience.

### ⚙️ Tech Stack

| Layer | Tools |
|-------|-------|
| **3D** | Three.js · React Three Fiber · Drei · Postprocessing |
| **Animation** | GSAP · ScrollTrigger |
| **UI** | React 19 · Tailwind CSS v4 · Vite 6 |
| **Contact** | EmailJS |

### 🔋 Features

- Animated 3D models (hero room, tech logos, contact computer)
- GSAP scroll-triggered section reveals and timeline animations
- Responsive layout — desktop, tablet, and mobile
- Live project showcase with custom artwork
- Animated stat counters (LeetCode, patents, accuracy)
- Contact form powered by EmailJS
- Highlights section (achievements, certifications, education)

## 🤸 <b><samp>Quick Start</samp></b>

**Prerequisites:** [Git](https://git-scm.com/) · [Node.js](https://nodejs.org/) · npm

```bash
git clone https://github.com/AryanSingh833/3d-portfolio.git
cd 3d-portfolio
npm install
```

Create a `.env` file in the project root:

```env
VITE_APP_EMAILJS_SERVICE_ID=
VITE_APP_EMAILJS_TEMPLATE_ID=
VITE_APP_EMAILJS_PUBLIC_KEY=
```

Replace the values with your [EmailJS](https://www.emailjs.com/) credentials, then:

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173/) to view the portfolio.
